Jux supports font family tokens from a list of available font families. If the font family you need isn't on the list, you can contact us on [Discord](https://discord.gg/BSXsahmyVQ) or email us at [squad@jux.io](mailto:squad@jux.io).

<CaptionImage caption="Font Family Token">
  <Image src={fontFamilyToken} alt="Font Family Token" />
</CaptionImage>

Font family tokens can be referenced by [Typography](/editor/tokens/typography) tokens in the font family field using aliases.

<CaptionImage caption="Font Family Alias">
  <Image src={fontFamilyAlias} alt="Font Family Alias" width={400} />
</CaptionImage>

Font family tokens consist of the following fields:

- Name (e.g., 'ui' or 'titles')
- Folder path (e.g., typography/core/titles)
- Value selection from a list
- Alias to another token - `optional`
- Description - `optional`

<CaptionImage caption="Font Family Props">
  <Image src={fontFamilyProps} alt="Font Family Props" width={563} />
</CaptionImage>

Font family tokens can only be used in the Typography module of the Dynamic Design Panel (DDP).

<CaptionImage caption="DDP Text Module">
  <Image src={ddpTextModule} alt="DDP Text Module" width={563} />
</CaptionImage>

To tokenize the font family field, click on the 'token icon button' that appears on the right side of the field when you hover over it.
