The layout module consists of the following submodules:

## Padding Submodule

Padding is the internal spacing inside an element, surrounding the content and appearing before the border. Units can be set in PX, VW, VH, EM, or REM.

<CaptionImage caption="Padding example">
  <Image src={padding} alt="Padding" width={463} />
</CaptionImage>

There are three control methods for padding:

- Unified (same padding on all sides)
- Pairs (top/bottom, left/right)
- Custom (individual control for each side)

<CaptionImage caption="Padding control methods">
  <Image src={paddingSides} alt="Padding units" />
</CaptionImage>

## Margin Submodule

Margin is the external spacing outside an element, surrounding the border. Units can be set in PX, VW, VH, EM, or REM.

<CaptionImage caption="Margin example">
  <Image src={margin} alt="Margin" width={352} />
</CaptionImage>

There are three control methods for margin:
- Unified (same margin on all sides)
- Pairs (top/bottom, left/right)
- Custom (individual control for each side)

<CaptionImage caption="Margin control methods">
  <Image src={marginSides} alt="Margin units" />
</CaptionImage>

## Display Submodule

The display submodule controls the layout of an element.

<CaptionImage caption="Display Submodules">
  <Image src={display} alt="Display" width={352} />
</CaptionImage>

Options include:

### Block

The element will take up the full width of the parent container and stack vertically.

<CaptionImage caption="Display Block">
  <Image src={displayBlock} alt="Display Block" width={352} />
</CaptionImage>

### Flex

The element will become a flex container, allowing you to control the layout of its children.

<CaptionImage caption="Display Flex">
  <Image src={displayFlex} alt="Display Flex" width={352} />
</CaptionImage>

Flex properties include:

#### Direction

Sets how flex items are placed in the flex container, and the direction (normal / reversed).

#### Wrapping

Sets whether flex items are forced onto one line or can wrap onto multiple lines. If wrapping is allowed, it sets the direction that lines are stacked.

<CaptionImage caption="Flex Wrapping">
  <Image src={flexWrapping} alt="Flex Wrapping" width={352} />
</CaptionImage>

#### Align Items

Sets the alignment of items along the cross axis, depending on the flex direction.

<CaptionImage caption="Align Items">
  <Image src={alignItems} alt="Align Items" width={452} />
</CaptionImage>

Align items can be set to 5 different options:

1. Start - Aligns items to the start of the container.
    <CaptionImage caption="Align Start">
      <Image src={alignStart} alt="Align Start" />
    </CaptionImage>

2. Center - Aligns items to the center of the container.
    <CaptionImage caption="Align Center">
      <Image src={alignCenter} alt="Align Center" />
    </CaptionImage>

3. End - Aligns items to the end of the container.
    <CaptionImage caption="Align End">
      <Image src={alignEnd} alt="Align End" />
    </CaptionImage>

4. Stretch - Stretches items to fill the container.
    <CaptionImage caption="Align Stretch">
      <Image src={alignStretch} alt="Align Stretch" />
    </CaptionImage>

5. Baseline - Aligns items to the baseline of the container.

#### Justify Content

Sets the alignment of items along the main axis, depending on the flex direction.

<CaptionImage caption="Justify Content">
  <Image src={justifyContent} alt="Justify Content" width={463} />
</CaptionImage>

Justify content can be set to 5 different options:

1. Start - Aligns items to the start of the container in the main axis.
    <CaptionImage caption="Justify Start">
      <Image src={justifyStart} alt="Justify Start" />
    </CaptionImage>

2. Center - Aligns items to the center of the container in the main axis.

3. End - Aligns items to the end of the container in the main axis

### Inline Flex

The element will become an inline flex container, allowing you to control the layout of its children. Inline flex has the same properties as flex.

### Inline

The element generates an inline box that does not create line breaks before or after itself. In normal flow, if there is available space, the next element will appear on the same line.

<CaptionImage caption="Display Inline">
  <Image src={displayInline} alt="Display Inline" />
</CaptionImage>

### None

Turning off the display of an object removes its effect on layout (the document renders as if the object doesn't exist). This setting also hides all descendant objects.

When an object is hidden:

- It cannot be accessed from the canvas
- It can only be accessed via the layers panel
- The layers panel will show a special 'eye' indicator for hidden objects

This feature allows for temporary removal of elements from the design without deleting them.
