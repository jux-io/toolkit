Installing Jux as PostCSS plugin is the recommended way to seamlessly use Jux in your project. This guide will walk you through the steps to set up Jux as a PostCSS plugin in your project.

<Steps>
    1. ###### Install Jux CLI
        <Tabs>
            <TabItem label={'npm'}>
                <Code code={`npm install -D @juxio/cli postcss`} title={'Terminal'} frame={'code'} />
            </TabItem>
            <TabItem label={'pnpm'}>
                <Code code={`pnpm add -D @juxio/cli postcss`} title={'Terminal'} frame={'terminal'} />
            </TabItem>
            <TabItem label={'yarn'}>
                <Code code={`yarn add -D @juxio/cli postcss`} title={'Terminal'} frame={'terminal'} />
            </TabItem>
        </Tabs>

    2. ###### Initialize a new project

        <Code code={`npx jux init --postcss`} title={'Terminal'} frame={'terminal'} lang={'shellscript'} />

        This will install the necessary dependencies, and create `jux.config.ts` and `postcss.config.js` files in your project root.

        <Code code={postcssConfig} lang="ts" title={'postcss.config.js'} />

    3. ###### Configure your include files

        Add the path to your source files in `jux.config.ts`. Jux will scan these files to generate the CSS file.
        <Code code={juxConfig} lang="ts" title={'jux.config.ts'} mark={3} />

    4. ###### Add Jux's @layer directive to your main CSS file.

        <Code code={`@layer juxbase, juxtokens, juxutilities;`} title={'main.css'} lang={'css'} />



    5. ##### Start using Jux in your project

        Include the generated CSS file in your project and start using Jux to style your components.
        <Code code={mainTsx} lang="tsx" title={'index.tsx'} mark={1} />

</Steps>
