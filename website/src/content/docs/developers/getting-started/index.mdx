This guide will walk you through the basics of setting up and using Jux in your project. Jux is a zero-runtime CSS-in-JS solution that leverages PostCSS to generate optimized stylesheets during build time.

#### Installation

The fastest way to get started with Jux is to use the Jux CLI. The CLI will help you set up a new project and generate the necessary configuration files.

<Steps>
    1. ###### Install Jux CLI
        <Tabs>
            <TabItem label={'npm'}>
                <Code code={`npm install -D @juxio/cli`} title={'Terminal'} frame={'terminal'} />
            </TabItem>
            <TabItem label={'pnpm'}>
                <Code code={`pnpm add -D @juxio/cli`} title={'Terminal'} frame={'terminal'} />
            </TabItem>
            <TabItem label={'yarn'}>
                <Code code={`yarn add -D @juxio/cli`} title={'Terminal'} frame={'terminal'} />
            </TabItem>
        </Tabs>


    2. ###### Initialize a new project

        This will install the necessary dependencies and create a `jux.config.ts` file in your project root.

        <Code code={`npx jux init`} lang="ts" title={'Terminal'} frame={'terminal'} />

    3. ###### Configure your include files
        Add the path to your source files in `jux.config.ts`. Jux will scan these files to generate the CSS file.
        <Code code={juxConfig} lang="ts" title={'jux.config.ts'} mark={3} />

    4. ###### Start Jux build process

        Jux CLI will scan your source files and generate a CSS file containing all the styles used in your project, along with tokens and themes configuration.
        <Code code={`npx jux generate -o ./src/jux.css`} title={'terminal'} />

    5. ##### Start using Jux in your project

        Include the generated CSS file in your project and start using Jux to style your components.
        <Code code={mainTsx} lang="ts" title={'main.tsx'} mark={1} />
</Steps>
