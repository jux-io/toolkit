Jux supports font family tokens from a list of available font families. If the font family you need isn't on the list, you can contact us on [Discord](https://discord.gg/xa4PR7T9) or email us at [squad@jux.io](mailto:squad@jux.io).

Font family tokens can be referenced by [Typography](/editor/tokens/typography) tokens in the font family field using aliases.

  <picture>
    <Image src={fontFamilyAlias} alt="Font Family Alias" width={400} />

Font family tokens consist of the following fields:

- Name (e.g., 'ui' or 'titles')
- Folder path (e.g., typography/core/titles)
- Value selection from a list
- Alias to another token - `optional`
- Description - `optional`

Font family tokens can only be used in the Typography module of the Dynamic Design Panel (DDP).

To tokenize the font family field, click on the 'token icon button' that appears on the right side of the field when you hover over it.
