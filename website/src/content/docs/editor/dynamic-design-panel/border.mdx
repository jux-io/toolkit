The Border module controls the border properties of an object and consists of two main submodules: Border Sides and Border Radius.

### Border Sides Submodule

This submodule includes a switcher with options for [`all` | `top` | `right` | `bottom` | `left` | `custom`], allowing precise control over each side of the border. It comprises:

1. Border Width: A dimension type field
2. Border Color: A color type field
3. Border Style: A simple selector field

These properties can be applied independently to each of the four sides.

### Border Radius Submodule

This submodule includes a switcher with options for [`all` | `top-left` | `top-right` | `bottom-right` | `bottom-left`], allowing precise control over each corner of the border.
